# muratogat.com

Personal website for **Murat Ögat** — Co-Founder & CTO at [Aktionariat AG](https://www.aktionariat.com).

Live at [muratogat.com](https://muratogat.com).

## Stack

Pure HTML, CSS, and vanilla JavaScript. No frameworks, no build step, no dependencies.

## Local Development

Open `index.html` in your browser. That's it.

## Deployment

Hosted on [GitHub Pages](https://pages.github.com/) with a custom domain via the `CNAME` file.
